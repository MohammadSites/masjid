# شاشة المسجد — مواقيت الصلاة

تطبيق لعرض مواقيت الصلاة على شاشة التلفزيون أو التابلت. يعمل **بدون إنترنت**.

## التحميل والتشغيل (بدون تثبيت أي أدوات)

- **تطبيق أندرويد (APK):** حمّل الملف الجاهز من **Actions** → آخر تشغيل **Build APK** → Artifacts → **masjid-screen-apk**. ثبّته على التلفزيون/التابلت وافتحه (يفتح تلقائياً عند تشغيل الجهاز).
- **نسخة فلاشة (أندرويد):** حمّل **masjid-flash-android** من نفس صفحة Actions، فك الضغط، انسخ المجلد إلى الجهاز، واتبع **FLASH_ANDROID.md**.

التفاصيل الكاملة: **OFFLINE_USE.md**.
