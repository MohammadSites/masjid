# تشغيل شاشة المسجد بدون إنترنت (مجاني)

لا حاجة لـ Vercel ولا لإنترنت بعد التثبيت.

---

## 1. تطبيق أندرويد (APK) — جاهز للتحميل والتثبيت

تطبيق واحد جاهز: ثبّته على التلفزيون أو التابلت وافتحه. **يفتح تلقائياً عند تشغيل الجهاز**. لا تحتاج تثبيت Node.js ولا Android Studio.

### تحميل ملف الـ APK

1. ادخل إلى مستودع المشروع على GitHub: **MohammadSites/masjid**
2. افتح تبويب **Actions** → اختر آخر تشغيل ناجح لـ **Build APK**
3. في أسفل الصفحة ستجد **Artifacts** → حمّل **masjid-screen-apk**
4. انقل ملف الـ APK إلى التلفزيون/التابلت (USB أو بريد أو تخزين سحابي) وثبّته
5. بعد التثبيت، عند إعادة تشغيل الجهاز سيفتح التطبيق من نفسه

**رابط مباشر (بعد تشغيل الـ workflow مرة واحدة):**  
من صفحة المستودع: **Actions** → **Build APK** → **Summary** → **Artifacts** → **masjid-screen-apk**

---

## 2. نسخة الفلاشة — للتشغيل على أندرويد من مجلد (بدون APK)

إذا أردت تشغيل التطبيق من **مجلد على الجهاز أو على فلاشة USB** (بدون تثبيت APK):

- **تحميل مجلد جاهز:** من **GitHub Actions** → آخر تشغيل لـ **Build APK** → Artifacts → حمّل **masjid-flash-android** (ملف zip). فك الضغط وانسخ المجلد إلى الأندرويد أو الفلاشة.
- أو انسخ مجلد التطبيق من المستودع (الملفات: `index.html`, `app.js`, `data/`, `media/`, …).
- اتبع الخطوات في **FLASH_ANDROID.md** (تطبيق HTTP Server + Fully Kiosk للتشغيل التلقائي عند فتح الجهاز).

كل شيء موضّح في **FLASH_ANDROID.md** — لا حاجة لـ Windows ولا لـ Node.js.
